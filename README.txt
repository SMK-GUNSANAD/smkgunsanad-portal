# Portal Digital SMK Gunsanad

**Bahasa Melayu (BM):**
Portal Digital SMK Gunsanad dibangunkan untuk memudahkan warga sekolah mengakses sistem utama sekolah seperti Sistem Jual Beli Jadual Waktu.
Portal ini menggunakan tema rasmi sekolah: hitam, kuning, hijau dan putih.
Fail ini boleh dimuat naik ke **GitHub Pages** untuk akses awam percuma.

**Langkah Pemasangan:**
1. Muat naik semua fail ke dalam repositori GitHub.
2. Pergi ke Settings → Pages → pilih `main branch / root` → Save.
3. Laman akan boleh diakses di pautan GitHub Pages sekolah.

---

**English Version:**
The SMK Gunsanad Digital Portal was developed to make it easier for the school community to access main systems such as the Teaching Schedule Exchange System.
The portal follows the official school theme colors: black, yellow, green, and white.
These files can be uploaded to **GitHub Pages** for free public access.

**Installation Steps:**
1. Upload all files to your GitHub repository.
2. Go to Settings → Pages → select `main branch / root` → Save.
3. Your portal will be live via the GitHub Pages link.
